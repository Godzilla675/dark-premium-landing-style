# Dark Premium Landing Page Style

A reusable AI skill for building premium, dark-themed, Apple-inspired landing pages.

Inspired by the visual style of [getcalendarly.com](https://getcalendarly.com/) — not a copy, but a distilled design system capturing the same aesthetic: pure black backgrounds, cinematic whitespace, scroll-triggered animations, and refined minimalism.

## What's inside

**[`AGENTS.md`](./AGENTS.md)** — The complete design system covering:

- Color palette (pure black, white opacity hierarchy, accent colors)
- Typography (Geist/Inter, type scale, spacing rules)
- Spacing system (8px grid, generous section padding)
- 7 component specs (nav, hero, CTAs, centered features, two-column features, floating icons, footer)
- Animation patterns (scroll fade-ins, marquee, hover effects)
- Gradient & overlay recipes (vignettes, glows)
- Responsive breakpoints
- Page structure & implementation checklist
- Copy tone guidelines
- Do's and don'ts

## Compatibility

Works with any AI coding tool that supports standard instruction files:

| File | Tool(s) |
|------|---------|
| `AGENTS.md` | OpenAI Codex, Cursor, Amp, Jules (Google), Factory, GitHub Copilot CLI |
| `CLAUDE.md` | Claude Code (imports AGENTS.md) |
| `GEMINI.md` | Gemini CLI |
| `.github/copilot-instructions.md` | GitHub Copilot (VS Code, GitHub.com) |
| `.cursor/rules/` | Cursor Rules |

## Usage

Clone or add this repo to your project, then ask your AI assistant to build a landing page — it will automatically pick up the style instructions from the appropriate file.
